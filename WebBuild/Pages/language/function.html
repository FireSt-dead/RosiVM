Function

Well its a function... Differs from the methods by being defined in a module instead of a class. I dislike C# static classes. Functions would be used for example in min, max etc. defined in module math.

Function definition:

<Function> ::=
    <Visibility> 'function' <Name> <TypeParameters> l-paren <Arguments> r-paren <ReturnType>
    <StatementBlock>

For example:

module math { function min(x: number) => number { } }

Be very careful not to write procedural code using functions. Classes are still good for you.
