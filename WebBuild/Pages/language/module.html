Module

Modules group common functionalities together. We have to elaborate on module namespaces, but for now modules will simply be referred by name.

Module syntax:

<Module> ::= 'module' <Name> l-brac <ModuleMembers> r-brac
<ModuleMembers> ::= <ModuleMember> <ModuleMembers> | <>
<ModuleMember> ::= <Function> | <Class> | <Interface>

For example:

module geometry { }
