Class

Classes encapsulate data with operations. Basic building block in Object Oriented Programming.

A class can extend a single base class to inherit its members.

A class may implement multiple interfaces.

Class definition:

<Class> ::=
    <Visibility> 'class' <Name> <TypeParameters>
    <Extends> <Implements>
    l-brac <ClassMembers> r-brac
<ClassMembers> ::= <ClassMember> <ClassMembers> | <>
<ClassMember> ::= <Field> | <Constructor> | <ClassMethod>

For example:

public class Point2D extends Point3D implements Clonable {
  // fields, constructos, methods etc.
}
