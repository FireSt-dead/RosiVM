RosiLang, The Language

Currently the language is under development so we will elaborate on examples later. For now this is a small preview:


public class Point {
  var x: number;
  var y: number;

  public method add(p: Point) => Point {
    this.x += p.x;
    this.y += p.y;
    return this;
  }
}
 

The language is inspired pretty much by the C# type system and AS3 and TypeScript syntax.

We will try to build the language so that APIs from common platforms such as Java and .NET can get mapped easily to RosiLang constructs and RosiLang should be able to compile to Java, .NET and JavaScript.
