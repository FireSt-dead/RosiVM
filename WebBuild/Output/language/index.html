RosiLang, The Language

Currently the language is under development so we will elaborate on examples later. For now this is a small preview:

  1. module geom {
  2.     public class Point {
  3.         var x: number;
  4.         var y: number;
  5.         public method add(p: Point) => Point {
  6.             this.x += p.x;
  7.             this.y += p.y;
  8.         }
  9.     }
  10. }

The language is inspired pretty much by the C# type system and AS3 and TypeScript syntax. We are currently adding some noise (such as the "method") keyword to avoid easily grammar ambiguity, but it probably would be removed in the final version.

Language Definition

The grammer is written in the GOLD language. GOLD uses a BNF as a language and can build a LALR parser, which is cool... and free... and integrates well with .NET...
