Interface

An interface defines a set of members a class may implement. A class may implement multiple interfaces. Interfaces play important role in defining behaviors, and simulating multiple inheritance.

Interface syntax:

<Interface> ::=
    <Visibility> 'interface' <Name> <TypeParameters>
    <InterfaceExtends>
    l-brac <InterfaceMembers> r-brac
<InterfaceExtends> ::= 'extends' <Types> | <>
<InterfaceMembers> ::= <InterfaceMember> <InterfaceMembers> | <>
<InterfaceMember> ::= <InterfaceField> | <InterfaceMethod>

For example:

  1. module model {
  2.     public interface Comparable {
  3.         var hash: number;
  4.         method isLessThan(other: Comparable) => boolean;
  5.         method isGreaterThan(other: Comparable) => boolean;
  6.     }
  7. }
