RosiVM - High Level Object Oriented Language and Virtual Machine.

After following various projects such as:

  • xmlvm
  • haxe
  • llvm
  • openfl

And inspired by some of the languages:

  • ActionScript 3
  • C#
  • Java
  • TypeScript

The RosiVM project will try to build infrastructure for a modern object oriented programming language. We will try to provide rich type system, but keep custom operators and types at minimum. For example we would avoid AS3's XML types, .NET's lync types, Java's nested classes, we would preserve modules, classes, interfaces, generic parameters with co and contra variance. We will also try to minimize compiler magic such as building fields from constructor parameters, yield return, async await etc.

Initially we will focus on the type system and language specification. We will use an XML to represent the language AST and probably keep the XML as intermediate representation. It can easily be traversed to generate bytecode for .NET using System.Emit (with debugging support). Also it should be able to generate LLVM-IR, but we will need GC. We can also convert the XML using XSLT to the XMLVM and have its toolchain cross compile.

In the long distant future we may build a custom virtual machine but that task would be too heavy. We will be making small steps for now.
